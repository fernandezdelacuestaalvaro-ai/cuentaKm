# Cuenta KM GPS

Aplicación Python para registrar rutas GPS, velocidad y distancia recorrida.

## Tecnologías

- Python
- Flask
- HTML
- JavaScript
- Geolocation API

## Funcionalidades

- Lectura GPS real desde navegador
- Velocidad en tiempo real
- Coordenadas GPS
- Preparado para exportar rutas CSV
- Compatible con móvil

## Ejecutar proyecto

Instalar dependencias:

```bash
pip install -r requirements.txt

```
Ejecutar:

python web_app.py

Abrir navegador:

http://127.0.0.1:5000
